Back2Mac (B2M)
=====

[Back to the Mac](http://macnews.tistory.com/) 블로그를 쉽게 이용하기 위해 제작된 앱입니다.

기존 앱이 오랫동안 작동하지 않아 만들기 시작했는데 하필 시작하자마자 기존 앱이 복구가 되었네요 ;(

처음부터 오픈소스로 공개하려 했으니, 스토어에 등록은 안하더라도 소스만이라도(..) 공개합니다!

서버 부분도 시간날 때 정리해서 추가로 반영하겠습니다.

## 기능
1. 새 글이 왔을 때 푸시 알림
2. 글 읽음 표시
3. 북마크(책갈피) 기능 <- 현재 미구현!
